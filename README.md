# 科技感 2D 射擊模擬訓練器
這是一個基於 Google Apps Script 與 Google Sheets 開發的 2D 模擬器。

### [🚀 點此開啟模擬器 (Live Demo)](https://script.google.com/macros/s/AKfycbxX4KQ0rR-YOOvZNtUxfePtZq-cxh2q9Y508fR8bdwDLrPHqoREVxDnFR3f-B4Zdex9/exec)

## 功能特色
- **科技感 UI**: 使用霓虹風格介面與動態音效。
- **即時排行榜**: 分數自動同步至 Google Sheets。
- **CRUD 系統**: 支援玩家註冊、紀錄讀取、分數更新與帳號刪除。
